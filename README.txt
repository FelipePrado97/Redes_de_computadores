Trabalho de Redes
Arquivos: Servidor.py, Cliente.py

Para executar basta: extrair os arquivos para o diretório desejado, executar o Servidor.py e o Cliente.py respectivamente para fazer a conexão!
Será Exibido em Cliente.py um Menu com as Seguintes Funcionalidades:
[1] - Criar Diretório
[2] - Remover Diretório
[3] - Listar Conteúdo de Diretório
[4] - Enviar Arquivo
[5] - Remover Arquivo
[6] - Sair

Basta digitar a opção desejada e seguir os passos que serão enviados pelo Prompt do Cliente.py.
